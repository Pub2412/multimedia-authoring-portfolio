Project Goal

Create a short animation sequence with realistic movement and hand-drawn artistic expression.

Main Tools

Adobe Premiere Pro, CLIPART STUDIO, and timeline-based compositing for output refinement.


Procedures:

  1. Record or select reference footage with stable movement.
  2. Break footage into key frames and define frame rate targets.
  3. Trace and stylize each frame while preserving motion continuity.
  4. Composite all frames, apply color treatment, and render final sequence.
